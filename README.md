# Asteroid Miner Game (Unity)

A 3D space mining game built in Unity where the player travels through dangerous asteroid fields, collects resources, and upgrades their suit.

## Overview
This project is a Unity game focused on mining asteroids in space while avoiding hazards and managing progression. The player gathers materials from asteroids, uses those materials to improve their ship, and gradually gains access to more dangerous and rewarding areas.

## Why I Built This
I built this project because I wanted to make a game that combined resource collection, progression, and horror in a space setting. It also gave me more experience with Unity, C#, and designing gameplay systems.

## Features
- Asteroid decceleration and drilling
- "Watcher" enemies (don't look at them for too long!)
- Basic ship to refill oxygen and health
- Player upgrades

## Tech Stack
- Unity
- C#

## Current Status
This project is currently a work in progress.

## How to Run
1. Clone the repository
2. Open the project in Unity
3. Load the main scene
4. Press Play in the Unity editor
